cd:Where Am I?
dir: What's Around me?
dir /a: Show the hidden files
cd .. : Take you back to previous directory
cd /d <path> : Go the path you tell.
dir /s <file>: find the file on the disk
type <file>: Allow you to read the sepific file.
whoami: Who logged in ?
hostname: What is the name of this computer.
systeminfo: provide you information about the windows.
ipconfig: How is this machine connected to the network such as ipv4 and ipv6 and default gateways
cls or clear: clear cmd
ver: Shows the operating system version.
systeminfo: Provides detailed system info like OS, processor, and memory.
driverquery | more: Lists installed drivers, showing one page at a time.
help: Displays help for specific commands.
ipconfig /all: Displays detailed network info, including DNS servers and DHCP status.
ping [target_name]: Tests network connectivity by sending ICMP packets.
tracert [target_name]: Traces the network route to a target, showing each hop along the way.
tree to visually represent the child directories and subdirectories.
mkdir stands for make directory
tasklist: We can list the running processes using tasklist.
chkdsk: checks the file system and disk volumes for errors and bad sectors.
driverquery: displays a list of installed device drivers.
taskkill PID <PID>: for kill the process.
tasklist /FI "imagename eq file_name": find the file.
sfc /scannow: scans system files for corruption and repairs them if possible.
