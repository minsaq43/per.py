pwd : stands for print working directory.
ls : show the list of directories.
ls -l: details of directories with name of directories with file sizes, premission and dates and more.
ls -al: show the hidden files in present directories.
cd : stands for change directory .cd <directory> for example: cd Documents
cd .. : back to previous directory.
find command is for finding a specific file in computer, for example: find ~ -name mission_brief.txt
cat : is for read a file.
whoami: show username
uname -a :tell on what system are you?
uname: operating system's name
df -h : check disk and storage information.
cat os-release: about he os.
